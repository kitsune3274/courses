Цель домашнего задания
Научиться работать с переменными и их типами, с переменными без значения и с числовыми данными.

Что нужно сделать
1.	Создайте переменную, которая будет хранить строку «Hello, Skillbox» и напечатайте её содержимое с помощью команды print().
2.	Создайте константу, которая будет хранить строку «Hello, Skillbox» и напечатайте её содержимое с помощью команды print().
3.	Создайте переменную с явным указанием типа, присвойте ей значение 0.5 и напечатайте её содержимое с помощью команды print().
4.	Возьмите optional-переменную ниже, извлеките из неё значение (раскройте optional) и напечатайте её значение. Результат не должен содержать обозначения optional. 

var swiftString: String? = "Swift"
// Ваш код 

5.	Напишите код, который складывает два числа: 10 и 0,5. Выведите результат командой print().

Советы и рекомендации
Рекомендуем прочитать статью про переменные и типы, чтобы лучше справиться с заданием и усвоить материал.

Что оценивается
Все задания должны быть выполнены, код должен работать.

Как отправить задание на проверку
Пришлите на проверку скриншот, на котором есть вывод результата работы.
